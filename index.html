Instalación rápida

  1. Descarga el archivo `RBP_Setup_2.0.0.exe`
  2. Ejecuta el instalador. Si aparece SmartScreen, pulsa Más información y luego Ejecutar de todas formas.
  3. Abre RBP desde el acceso directo
  4. Activa con tu clave de licencia

¿Por qué aparece SmartScreen? Windows usa ese aviso cuando una app nueva aún no tiene suficiente reputación de descargas. No significa que sea malware por sí solo. Para continuar de forma segura, valida siempre que descargaste desde esta página oficial o desde el release oficial del proyecto.

Preguntas frecuentes

Windows SmartScreen muestra advertencia, ¿es normal?

Sí. Puede aparecer en algunos equipos cuando la aplicación es nueva. SmartScreen protege contra archivos desconocidos. Si descargaste desde el canal oficial, sigue estos pasos:

1) Clic en Más información.
2) Clic en Ejecutar de todas formas.
3) Si Windows pregunta permisos, aceptar para continuar.

¿Dónde consigo mi clave de licencia?

Escríbenos por WhatsApp o correo con tu Machine ID (visible en la pantalla de activación).

¿La app funciona sin internet?

Sí. La app funciona localmente en tu PC. Internet solo se usa para descarga, activación y actualizaciones.
